# FinGuard AI

FinGuard AI is an AI-powered financial fraud detection system that analyzes transactions and identifies suspicious activities in real time.

## Features
- Machine learning fraud detection
- Interactive Streamlit dashboard
- Fraud risk visualization
- Fraud simulation tool

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Plotly

## Run the Project

Install dependencies:
pip install -r requirements.txt

Run the app:
streamlit run app.py
